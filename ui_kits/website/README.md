# Website UI Kit — ChristianWS

Interactive prototype of the personal resume/portfolio website for Christian Wyterlin Silveira.

## Sections
1. **Profile (Sobre)** — Hero with illustration, intro text
2. **Experience (Experiências)** — Job cards on white bg
3. **Education (Formação Acadêmica)** — Academic details on dark bg
4. **Technologies** — Logo grid with hover dim effect
5. **Contact (Contato)** — Floating-label form on gradient bg
6. **Footer** — Brand, links, social

## Typography
- Headings: Montserrat Bold, UPPERCASE
- Body: Open Sans, Regular

## Color Palette
- Black `#000000`, Royal Blue `#4169E1`, White `#ffffff`, Dark `#0d1117`, Gold `#C9A84C`

## Notes
- No real SVG tiled backgrounds (assets not in repo), substituted with CSS textures
- Technology PNG logos not available — substituted with styled text badges
- Form is non-functional (EmailJS not wired)
