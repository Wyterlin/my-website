# Meu Site Pessoal - Currículo Online

Bem-vindo ao repositório do meu site pessoal! Este projeto tem como objetivo apresentar meu perfil profissional, minhas experiências, habilidades e projetos de forma organizada e interativa.

## ✨ Sobre o Projeto

Este site serve como um currículo digital, permitindo que recrutadores e interessados conheçam mais sobre minha trajetória profissional e minhas competências técnicas. Ele inclui seções como:

- **Sobre Mim**: Breve apresentação e meus objetivos profissionais.
- **Habilidades**: Tecnologias e ferramentas que domino.
- **Experiência**: Meus trabalhos anteriores e projetos relevantes.
- **Projetos**: Portfólio com links para repositórios no GitHub e demonstrações.
- **Contato**: Links para redes sociais e formulário de contato.

## 🛠 Tecnologias Utilizadas

O site foi desenvolvido utilizando:

- **HTML5**
- **CSS3**
- **JavaScript**
- **React**
