# Meu Site Pessoal - Currículo Online

Bem-vindo ao repositório do meu site pessoal! Este projeto tem como objetivo apresentar meu perfil profissional, minhas experiências, formações e tecnologias que utilizo.

## ✨ Sobre o Projeto

Este site serve como um currículo digital, permitindo que recrutadores e interessados conheçam mais sobre minha trajetória profissional e minhas competências técnicas.

## 🛠 Tecnologias Utilizadas

O site foi desenvolvido utilizando:

- **HTML5**
- **CSS3**
- **JavaScript**
- **React**

Espero que este projeto ajude a mostrar um pouco mais sobre mim e meu trabalho. Fique à vontade para explorar e deixar feedback! 🚀
